# Switch Vision SNMP2MQTT

The SNMP polling and MQTT publishing backend for Switch Vision.

This project is based on SNMP2MQTT by Daniel Chesterton and Andrew J. Swan. It consumes standard SNMP2MQTT target YAML, including the review-only configuration generated by Switch Vision Discovery.

## Build

```bash
yarn install --frozen-lockfile
yarn build
```

## Run

Place `config.yml` in the working directory, then run:

```bash
yarn start
```

The Home Assistant add-on wrapper is maintained separately in `switch-vision-snmp2mqtt-addon`.

## Vendor architecture

Switch Vision is vendor-neutral. Vendor-specific helpers belong under `src/vendors/<vendor>/`. Cisco remains the first supported vendor and is represented under `src/vendors/cisco/`.
